# Customer Churn Prediction

## 📌 Description
This project predicts whether a customer will leave the company (churn) or stay using Machine Learning.

## 🚀 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Flask

## 📂 Project Structure

Customer-Churn-Prediction/
│── app.py
│── data.csv
│── dataset.py
│── main.py
│── predict.py
│── preprocessing.py
│── requirements.txt
│── train_model.py
│── README.md

## ⚙️ Features
- Data preprocessing
- Model training
- Churn prediction
- Flask web deployment

## ▶️ Run Project

pip install -r requirements.txt

python train_model.py

python app.py
