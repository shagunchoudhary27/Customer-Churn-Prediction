## 🚀 Overview
# Customer Churn Prediction Project
Customer churn is one of the biggest challenges for banks and subscription-based businesses. This project builds a machine learning model to predict whether a customer is likely to leave (churn) or stay based on customer profile, account details, and service usage.

## Overview
This project predicts whether a customer will leave the bank or stay.

 📌 File Description
 ---
 | File Name          | Purpose                              |
| ------------------ | ------------------------------------ |
| `data.csv`         | Customer churn dataset               |
| `dataset.py`       | Loads and displays dataset           |
| `preprocessing.py` | Checks missing values and data shape |
| `train_model.py`   | Trains ML model and saves model      |
| `predict.py`       | Predicts churn using saved model     |
| `main.py`          | Main project runner/instructions     |
| `app.py`           | Flask web app for deployment         |
| `model.pkl`        | Trained machine learning model       |
| `requirements.txt` | Required Python libraries            |
| `README.md`        | Project documentation                |
