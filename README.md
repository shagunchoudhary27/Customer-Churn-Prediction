# Customer Churn Prediction Project

## Overview
This project predicts whether a customer will leave the bank or stay.

## Files
- data.csv
- preprocessing.py
- train_model.py
- predict.py
- app.py

## Technologies Used
- Python
- Pandas
- Scikit-learn
- Flask

## Run Project

pip install -r requirement.txt

python train_model.py

python predict.py

python app.py
