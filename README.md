## 🚀 Overview

Customer churn is one of the biggest challenges for banks and service-based companies. This project builds a **machine learning model** to predict whether a customer is likely to leave the bank (churn) or remain active based on customer account details, demographics, and banking behavior.

The project includes:

* Data preprocessing
* Customer dataset analysis
* Model training using Logistic Regression
* Customer churn prediction
* Basic Flask web application
* Deployment-ready project structure

##🖥️ How to Run the Project
1️⃣ Install Required Libraries

3️⃣ Preprocess Data

python train_model.py
5️⃣ Predict Customer Churn
python predict.py
